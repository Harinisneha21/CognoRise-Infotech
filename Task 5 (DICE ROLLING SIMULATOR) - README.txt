This Dice Rolling Simulator is a Python program that emulates rolling dice. Users can specify the number of sides on the dice and the number of rolls they want to simulate. The program utilizes random number generation to simulate each roll and displays the results to the user. It provides a quick and enjoyable way to experience the randomness of dice rolls, commonly encountered in various tabletop games and board games. The simulator offers users the opportunity to explore the outcomes of dice rolls without the need for physical dice, making it a convenient tool for gaming enthusiasts.
